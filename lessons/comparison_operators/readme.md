# Comparison Operators

Comparison operators are used to compare two values.

## Operators

- `==` : Equal to
- `!=` : Not equal to
- `>` : Greater than
- `<` : Less than
- `>=` : Greater than or equal to
- `<=` : Less than or equal to

## Examples

```python
# Equal to
print(10 == 10)  # Output: True

# Not equal to
print(10 != 5)  # Output: True

# Greater than
print(10 > 5)  # Output: True

# Less than
print(10 < 5)  # Output: False
```
