# Bitwise Operators

Bitwise operators are used to perform bit-level operations.

## Operators

- `&` : AND
- `|` : OR
- `^` : XOR
- `~` : NOT
- `<<` : Zero fill left shift
- `>>` : Signed right shift

## Examples

```python
# AND operator
print(5 & 3)  # Output: 1

# OR operator
print(5 | 3)  # Output: 7

# XOR operator
print(5 ^ 3)  # Output: 6
```
